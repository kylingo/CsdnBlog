# 项目简介
这是一款可以随时查看CSDN博客的开源软件，欢迎Star或Fork。
默认支持Android分类的博客，内置了五十余名Android博客专家。

最新版本v2.1.5
- [点我下载](http://www.pgyer.com/csdn)
- [更新历史](./app/Release.md)

## 功能列表
- 查看博客专家列表
- 查看博客列表、分类、专栏、详情、评论
- 支持自定义添加、删除、置顶博主
- 支持博客收藏、历史、分享
- 支持离线阅读
- 支持最新、最热的博客
- 支持软件自动更新，升级
